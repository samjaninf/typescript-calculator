> A calculator built with [TypeScript](https://github.com/Microsoft/typescript).

## Live Demo

https://typescript-calculator.surge.sh

## Setup

```sh
$ git clone https://github.com/wyze/typescript-calculator.git
$ cd typescript-calculator
$ yarn # or `npm install`
```

## Development

```sh
$ npm run dev
```

## Build

```sh
$ npm run build
```
